# Product Vision - Todo List (Workflow Test)

## Why this exists

This is a **test project** to validate our AI agent workflow
(discovery -> product docs -> PRD -> vertical slice -> review -> handoff) on
something small and real. The product itself is intentionally tiny.

## Vision

The simplest possible personal todo list: a single user jots down tasks, checks
them off, and removes them - entirely in the browser, with no accounts and no
setup.

## What success looks like

- A first-time user understands the app in seconds, with no instructions.
- Create / view / complete / delete all work and feel instant.
- Tasks are still there after closing and reopening the browser tab.
- The whole thing runs by opening one file - no server, no install.

## Non-goals

Accounts, collaboration, sync, mobile apps, reminders, or anything that turns a
to-do list into a project manager. If a feature isn't needed to prove the
workflow, it is out.
